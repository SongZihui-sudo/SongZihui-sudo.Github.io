# 原码，反码还有补码
## 进制基数(radix)

二进制 0b 0B

八进制 H 0o

十六进制 0x

## 原码

最高位为符号位，0表示正数，1表示负数。

正数的原码与其自身相同，负数的只是符号位不同

```c
正数 a= 5 						原码：0101

负数b=-5						原码二进制：1101
```

## 反码

最高位为符号位，0表示正数，1表示负数。

负数除了符号位全部取反。

正数的反码与其自身相同。

```
正数 a= 5 						原码：0101

负数b=-5						原码二进制：1010
```

## 补码

最高位为符号位，0表示正数，1表示负数。

正数的补码等于本身，负数的补码等于反码+1

```
正数 a= 5 						原码：0101

负数b=-5						原码二进制：1011
```

**负数的补码，是能够和其相反数相加通过溢出从而使计算机内计算结果变为0的二进制码**。

**正十进制数补码等于其本身，n位寄存器下-X的补码等于**![[公式]](https://www.zhihu.com/equation?tex=2%5En-X)**对应的二进制编码。**

补码的补码就是原码
$$
2^{n} - X_{补}
$$
